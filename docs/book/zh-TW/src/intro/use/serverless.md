# 公開雲端中的 Serverless 函式即服務

WasmEdge 與現有的 Serverless 或 Jamstack 平台配合使用，為函式提供高效能、可移植和安全的 runtime 。即使在這些平台的 Docker 或 microVM 中執行，也能提供顯著的優勢。
