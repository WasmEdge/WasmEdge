# Kubernetes

大多数高级容器运行时都实现 Kubernetes 的 CRI（容器运行时接口）规范，以便可以由 Kubernetes 工具管理。 这意味着你可以使用 Kubernetes 工具来管理 pod 和命名空间中的 WebAssembly 应用程序映像。
