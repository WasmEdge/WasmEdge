pub struct TableType { /* private fields */ }
Expand description

Parses in-memory bytes as either the WebAssembly Text format, or a binary WebAssembly module Struct of WasmEdge TableType.

A TableType is used to declare the element type and the size range of a WasmEdge Table to be created.
